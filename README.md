# 📊 YouTube Watch History Analysis

A Data Science project that analyzes personal YouTube watch history data using Python, Pandas, NLP techniques, and data visualization.

This project explores behavioral patterns, content interests, watch-time trends, and user consumption habits from exported YouTube watch history JSON data.

---

# 🚀 Features

- Data Cleaning & Preprocessing
- Time-based Activity Analysis
- Top Watched Channels
- Hourly & Daily Usage Trends
- Content Interest Categorization
- WordCloud Visualization
- Behavioral Analytics
- Exploratory Data Analysis (EDA)

---

# 🛠️ Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
- Jupyter Notebook

---

# 📂 Dataset

The dataset was exported from Google Takeout in JSON format.

Dataset fields include:

- Video Title
- Channel Name
- Watch Timestamp
- Video URL
- Activity Type

---

# 📈 Analysis Performed

## 1. Data Cleaning

- Extracted channel names from nested JSON
- Converted timestamps into datetime format
- Removed invalid/missing entries
- Cleaned video titles

---

## 2. Time-Based Analysis

- Videos watched per day
- Hourly watch activity
- Peak usage patterns
- Daily engagement trends

---

## 3. Channel Analysis

- Most watched channels
- Frequency distribution
- Top content creators

---

## 4. Interest Analysis

Videos were categorized into topics such as:
